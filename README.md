# Test - REPO

To Clone
```
$ git@github.com:Akanshgarg8/test-repo.git
```

To Push

```
$ git push
```

To Commit

```
$ git commit -m "Test Message"
```

To See Git Log

```
$ git log
```


To Checkout to New Branch
```
$ git checkout -b new_branch
```

To Push New Branch to Origin
```
$ git push --set-upstream origin new_branch
```
